# Languages

프로그래밍 언어별 기초 문법, 개념, 학습 기록을 정리하는 공간입니다.

각 언어는 `theory/`와 `notes/`를 중심으로 관리합니다.

- `theory/`: 언어 개념과 문법을 정제한 문서
- `notes/`: 강의, 책, 학습 중 작성한 기록
