## 디렉토리 구조

```text
jump_to_python/
├── README.md
├── note/    # 학습 기록 Markdown
└── src/     # 실행 코드와 실습 파일
```

## 작성 원칙

- 학습 기록은 `note/<slug>.md`에 작성한다.
- 관련 코드는 `src/<slug>/`에 둔다.
- note와 src는 대소문자를 포함해 slug가 정확히 같을 때만 연결한다.
- 아래 학습 기록 marker 사이 내용은 자동 관리되므로 직접 수정하지 않는다.
